## WORKSHOP 3

โค้ดนี้เป็นการสร้างหน้า My Profile ด้วย React Native
เอาไว้แสดงข้อมูลส่วนตัวของผู้ใช้งานในรูปแบบแอปมือถือ

ภายในหน้าแอปจะแบ่งออกเป็นหลายส่วน ได้แก่
ส่วนหัวจะแสดงชื่อหน้า My Profile
ถัดมาจะเป็นกล่องข้อมูล 3 กล่อง คือ รหัสนักศึกษา คณะ และสาขา
ซึ่งจัดเรียงให้อยู่ในบรรทัดเดียวกันเพื่อให้ดูเป็นระเบียบ

ในส่วนของ ข้อมูลส่วนตัว การศึกษา และที่อยู่
ใช้ FlatList ในการแสดงข้อมูลเป็นรายการ
ทำให้โค้ดดูเป็นระบบ และสามารถเพิ่มหรือลดข้อมูลได้ง่าย

สำหรับหัวข้อ สิ่งที่ชอบ และ สิ่งที่ไม่ชอบ
จะใช้ Bullet List เพื่อแสดงรายการพร้อมจุดนำหน้า
ช่วยให้แยกแต่ละข้อออกจากกันได้ชัดเจน และดูอ่านง่าย

โดยรวมแล้วโค้ดนี้ช่วยให้หน้าโปรไฟล์มีโครงสร้างชัดเจน
ใช้งานง่าย และสามารถนำไปปรับแก้หรือพัฒนาต่อได้ในอนาคต
